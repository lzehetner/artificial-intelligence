Machine Learning

Machine Learning is a sub-field of artificial intelligence that enables systems to learn and improve automatically from experience (data) without being explicitly programmed

Machine learning can learn the following tasks
  • Value prediction make decisions on the basis of the analysed data (e.g. electricity consumption or sales forecast)

  • Calculation of probabilities for certain events (bps. purchase probability or termination probability)

  • Recognise groups and clustering in one data set

  • Recognition of interrelationships in sequences

  • Reduction of dimensions without much loss of information

  • Optimization of business processes

Deep Learning is a special method for processing information and a sub-area of Machine Learning and Artificial Intelligence. Deep Learning verwendet zur Analyse großer Datensätze neuronale Netze. Diese von Deep Learning angewandte Funktionsweise agiert ähnlich wie das menschliche Gehirn. Dabei werden Daten zuerst extrahiert, anschließend analysiert, um im Anschluss eine Schlussfolgerung bzw. Prognose zu erstellen. In practice, deep learning is mainly used to recognise images, understand texts or make better decisions
